Why these tools exist

Formatting a daily or weekly performance update for WhatsApp by hand is repetitive: bolding headings, adding the right emoji for a rank-1 vs a top-10 mention, inserting dividers between sections. The WhatsApp Formatter automates that pattern-matching so a report can go from plain text to a polished, ready-to-send message in one click.

The Topper Congrats Builder extends this to a specific recurring use case — congratulating branches or teams that top a leaderboard (registrations, sign-ups, etc.) across multiple categories like Yono registrations or WhatsApp Banking registrations, with support for month-to-date (MTD) toppers and multiple branches tying for the same rank.

The PDF Splitter solves a different but related problem: turning a multi-page PDF report into individual image files, which is often what's actually needed for sharing a single page on WhatsApp, Slack, or email rather than the whole document.

How it works

  • Everything runs client-side in JavaScript — no file or text you enter is uploaded to a server
  • The WhatsApp formatter applies pattern rules to plain text (rank mentions, headings, numbered lists) and outputs WhatsApp markdown syntax (asterisks for bold, underscores for italics)
  • The PDF splitter uses PDF.js to render each page to a canvas, auto-crops the whitespace border, and packages the results with JSZip for download
